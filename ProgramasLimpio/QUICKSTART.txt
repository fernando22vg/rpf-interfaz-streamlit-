  CACHING YA IMPLEMENTADO EN CÓDIGO:

  ┌─────────────────────────────────────────┐
  │ @st.cache_data(ttl=60)    ← LÍNEA ~183 │
  │ def _listar_archivos_cache(...):        │
  │     """Cache para escaneos FS"""        │
  │                                          │
  │ @st.cache_data            ← LÍNEA ~220 │
  │ def _load_tech_map(path):               │
  │     """Cache para Excel loads"""        │
  │                                          │
  │ @st.cache_data            ← LÍNEA ~234 │
  │ def _load_pmax_cargado(...):            │
  │     """Cache para data extraction"""    │
  └─────────────────────────────────────────┘

  RECOMENDACIONES ADICIONALES:
  • @st.cache_resource para DB connections (si aplica)
  • Limitar DataFrame size: df.head(1000)
  • Renderizado lazy (cargar bajo demanda)

  STATUS: ✅ OPTIMIZADO - caché ya presente
