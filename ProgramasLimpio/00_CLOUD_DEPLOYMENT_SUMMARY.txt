💾 ¿Y LOS DATOS?
═══════════════════════════════════════════════════════════════════════════════

¿Dónde están los datos?

OPCIÓN 1: En GitHub (archivos pequeños)
  - Ventaja: Simplemente funciona
  - Desventaja: No para datos > 100 MB

OPCIÓN 2: Descargar en tiempo de ejecución
  - Google Drive, OneDrive, API
  - Ventaja: Datos siempre frescos
  - Desventaja: Requiere acceso externo

OPCIÓN 3: Base de datos remota
  - PostgreSQL, MySQL en la nube
  - Ventaja: Escalable, seguro
  - Desventaja: Extra setup

RECOMENDACIÓN PARA TI:
  - Si datos < 100 MB: Incluir en GitHub
  - Si datos > 100 MB: Usar Google Drive o BD remota
